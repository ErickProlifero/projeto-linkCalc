# 🛰️ LinkCalc FTTH - Simulador de Viabilidade Óptica

O **LinkCalc** é uma ferramenta de suporte técnico desenvolvida para otimizar o processo de viabilidade em instalações de fibra óptica (FTTH). O projeto automatiza o cálculo de atenuação de sinal, considerando a perda por metragem do cabo drop e margens de segurança para conectores.

Este projeto une minha experiência prática como **Auxiliar de Telecom** com meus estudos em **Desenvolvimento Web**.

## 🚀 Funcionalidades
- **Cálculo de Atenuação:** Baseado na perda média de 0.35dB/km (fibra G.657).
- **Indicador de Status:** Feedback visual instantâneo (Viável ou Crítico) conforme padrões de mercado (-15dBm a -25dBm).
- **Interface Responsiva:** Otimizado para uso em campo via smartphone por técnicos instaladores.

## 🛠️ Tecnologias Utilizadas
- **HTML5 Semântico:** Estrutura focada em acessibilidade.
- **CSS3 Moderno:** Uso de Flexbox, Variáveis e Animações.
- **JavaScript:** Lógica de negócio para cálculos de engenharia de redes.
- **FontAwesome:** Ícones para melhor experiência do usuário (UX).

## 📈 Impacto no Negócio
Em um cenário real, esta ferramenta ajuda a:
1. Reduzir o retrabalho de equipes de campo.
2. Garantir que o cliente final receba o sinal dentro dos níveis de qualidade (KPIs).
3. Agilizar a tomada de decisão do técnico durante a instalação.
